# ShoppingListApp (Gestor de Lista de Compras)

Aplicación que gestiona listas de compras, desarrollado con Ionic Angular.

## Descripción

La aplicación permite a los usuarios crear y gestionar múltiples listas de compras. Los ítems se pueden categorizar (Aseo, Comida, etc.), marcar como completados y las listas pueden ser guardadas para ser reutilizadas.

## Cómo levantar el proyecto

**Requisitos:**
* Node.js
* Ionic CLI (`npm install -g @ionic/cli`)

**Pasos de ejecucion:**

1.  **Clonar el repositorio (o descargar el zip):**
    ```bash
    git clone https://github.com/ValLR/ShoppingApp.git
    ```

2.  **Entrar a la carpeta del proyecto:**
    ```bash
    cd ShoppingListApp
    ```

3.  **Instalar las dependencias:**
    ```bash
    npm install
    ```

4.  **Ejecutar la aplicación en el navegador:**
    ```bash
    ionic serve
    ```
